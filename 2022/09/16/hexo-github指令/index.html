hexo+github指令

发表时间：2022-09-16 阅读： 评论：

装逼必备

总结一下,github加hexo搭建博客的指令

  1. 安装git git教程

git --version检查是否安装成功

如果是linux系统的话，可以直接代码 sudo apt-get install git

有git的话建议使用git bash使用指令

  1. node js下载

下载完成的话可以使用 node -v npm -v 检测你的版本号，也可以看是否安装成功

  1. 去Github 上创一个github的号

  2. 选择一个地方去做你的仓库，在仓库的位置使用Git GUI里面的Help的show ssh key
    如果你的电脑没有ssh就可以点击Generate Key生成你的ssh

然后打开GitHub点到你的setting

new ssh key里面的title随便填，下面那就把生成的ssh放进去

  1. 安装hexo
    在仓库的位置右键先可以建一个blog，打开blog，在blog内右键打开git bash here执行一下指令

  • hexo -v检查是否安装完成

  • npm install -g hexo-cli

  • hexo init

  • hexo g

  • hexo s 这里会出现一个网址，直接浏览器输入，你就可以看到部署之后的样子ctrl+c结束

  • git config --global user.name "github的账号名"
    git config --global user.email "电子邮箱" 这两个代码就可以连上你的GitHub

  • ssh -T git@github.com 查看你的ssh有没有配置成功,成功了会回复你一个 Hi XXXXXX!you’ve successfully XXXXXXXXXXX

  1. 修改配置文件.config.yml 这里的repository你要填你自己的github的仓库名，别直接复制粘贴。分支也得用自己的分支名
    再安装一下.deploy-git

  • npm install hexo-deployer-git --save

  • hexo clean

  • hexo g;hexo s;hexo d; 部署完成，可以自己去看看自己的博客

  • hexo new "创建的文章名" 这是创建文章，来写自己的博客。

hexo主题 可以选一些自己喜欢的主题去搭建自己的博客

github切换hexo主题

打开_config.yml文件修改theme: liveforcode

将主题文件放到themes文件夹下面
